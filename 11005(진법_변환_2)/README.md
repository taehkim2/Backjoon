[아이디어]

10진법 숫자 N을 B진법으로 변환하려면, N이 B보다 작을떼 까지 B로 계속해서 나누어 준다.

N이 B보다 작아졌을때, 마지막 N과 모든 나머지들을 역순으로 나열하면 B진법으로 변환이 완료된다.

[풀이과정]

25를 2진법으로 표현해본다고 가정한다.

25 / 2 = 12,
25 % 2 = 1

12 / 2 = 6,
12 % 2 = 0

6 / 2 = 3,
6 % 2 = 0

3 / 2 = 1,
3 % 2 = 1

B보다 작은 N부터 모든 나머지들을 역순으로 나열하면 "11001" 이라는 2진법 숫자가 만들어진다.
