1874 - 스택 수열
==========

[개요]
--
> * 수열이 주어지고, 1 ~ n가지의 수를 스택에 넣었다가 늘여놓음으로써 주어진 수열을 만들 수 있는지 계산하는 프로그램을 작성하라.
> * 첫째 줄에는 숫자 N이 주어진다.
> * 둘째 줄부터는 숫자 N만큼의 수들이 주어진다.
> * 둘째 줄부터 마지막까지의 순서가 주어진 수열이다.
> * 스택에는 오름차순으로 N만큼 push가 이루어진다.
> * push / pop 은 "+" 와 "-" 기호로 출력하고, 가능하지 않은 경우는 "NO"를 출력한다.

[아이디어]
--
> * 주어진 수열을 동적 배열에 넣어둔다.
> * 동적 배열을 끝까지 반복하면서 조건에 따라 "+", "-", "NO"를 출력하도록 한다.

[풀이방법]
--
> 1. N + 1만큼의 int형 배열을 동적할당 하고, idx = 1 부터 for문을 돌리면서 수열을 입력받는다.
> 2. stack을 선언하고, stack의 top과 받아온 int형 배열의 값을 비교하면서 배열의 끝에 닿을 때 까지 while문을 반복한다.
> 3. stack이 비어있지 않고 && stack의 top과 arr[idx]가 같다면 -> res 문자열에 "-\n"을 추가하고 stack을 pop하고 idx를 1더해준다.
> 4. 위의 조건에 해당하지 않고 && num이 N보다 작거나 같다면 -> res 문자열에 "+\n"을 추가하고 stack에 num을 push하고 num을 1더해준다.
> 5. 위의 두 조건에 해당하지 않고 && stack이 비어있지 않고 && stack의 top과 arr[idx]가 다르다면 -> res 문자열을 "NO\n"로 초기화 하고 반복문을 빠져나온다.
> 6. res 문자열을 출력한다.
